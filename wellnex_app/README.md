# Wellnex - Flutter Fitness App

A production-ready Flutter fitness/step tracking app with gamification and rewards.

## 🚀 Features

- **Step Tracking**: Daily steps with progress rings
- **Analytics**: Weekly/monthly charts and insights  
- **Rewards System**: Points, streaks, achievements
- **Ad Monetization**: Rewarded ads with cooldowns
- **Material 3 UI**: Green/sky blue fitness theme

## 📱 Screens

1. Splash Screen
2. Onboarding (3 slides)
3. Login/OTP
4. Home Dashboard
5. Step Analytics
6. Rewards & Wallet
7. Ads Reward
8. Profile & Settings

## 🛠️ Setup

```bash
flutter pub get
flutter run
```

## 📦 Architecture

- **Clean Architecture + MVVM**
- **Riverpod** for state management
- **GoRouter** for navigation
- **Dio** for API calls
- **Hive** for local storage

## 🎨 Theme

Green (#10B981) + Sky Blue (#0EA5E9) color scheme.
